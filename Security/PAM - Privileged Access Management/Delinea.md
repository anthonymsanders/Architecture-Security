## 1. Product Capabilities

### Secret Server

| Capability | Details |
|---|---|
| Credential Vaulting | Military-grade AES-256 encrypted vault for passwords, SSH keys, and secrets |
| Automated Discovery | Scan networks and Active Directory for privileged accounts |
| Password Rotation | Automated, policy-driven password rotation across systems |
| Session Launcher | Launch RDP/SSH sessions with credential injection from the vault |
| Session Recording | Record and audit privileged sessions |
| Access Request Workflows | Approval-based checkout with time-limited access |
| Role-Based Access Control | Granular permissions with customisable roles and folder structures |
| Reporting & Audit | Comprehensive audit trail with compliance-ready reports |
| Platforms | Windows, Linux, Unix, databases, network devices, cloud platforms |
| Deployment | On-premises (Windows/IIS/SQL) or Cloud (SaaS) |

### Privilege Manager

| Capability | Details |
|---|---|
| Least Privilege | Remove local admin rights from endpoints |
| Application Control | Allowlisting, denylisting, greylisting, and reputation-based control |
| Just-in-Time Elevation | Elevate specific applications without granting full admin |
| Self-Service Elevation | End users request elevation with business justification |
| Platforms | Windows, macOS |
| Deployment | Agent-based; managed from centralised web console |

### Server PAM

| Capability | Details |
|---|---|
| Privilege Elevation | Just-enough privilege for Unix/Linux server administration |
| MFA at Login | Require multi-factor authentication for server access |
| Session Auditing | Command-level auditing of Unix/Linux sessions |
| AD Bridging | Authenticate Unix/Linux users via Active Directory |

### DevOps Secrets Vault (DSV)

| Capability | Details |
|---|---|
| Cloud-Native Secrets | Secrets management purpose-built for CI/CD pipelines |
| API-First | RESTful API and CLI for automation |
| Dynamic Secrets | Generate short-lived credentials on demand |
| Integrations | Jenkins, GitHub Actions, Azure DevOps, Terraform, Kubernetes, Ansible |

### Additional Products

| Product | Capability |
|---|---|
| **Account Lifecycle Manager** | Automated service account governance — discover, manage, decommission |
| **Connection Manager** | Centralised RDP/SSH session launcher with credential injection |
| **Delinea Platform** | Unified SaaS platform combining all modules |
